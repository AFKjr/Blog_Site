# Blog Site

A simple, interactive blog application built with vanilla HTML, CSS, and JavaScript. This project serves as a personal learning journal and blog platform.

## Features

- **Days Counter**: Tracks and displays the number of days since starting to learn JavaScript
- **Blog Posts**: Create, read, update, and delete blog posts
- **Auto-Save Drafts**: Automatically saves draft content to prevent loss
- **Local Storage**: Persists blog posts and drafts in the browser's local storage
- **Responsive Design**: Clean, modern UI that works on different screen sizes

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## Project Structure

- `index.html`: Main HTML file with the UI
- `app.js`: JavaScript logic for blog functionality and days counter
- `README.md`: This file
